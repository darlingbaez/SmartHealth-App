# SmartHealth

## Sistema de Gestión de Citas Médicas Online

### Descripción
SmartHealth es una aplicación móvil desarrollada para facilitar la gestión de citas médicas entre pacientes, médicos y centros de salud.

### Tecnologías utilizadas
- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore
- Firebase Storage
- GitHub

### Arquitectura
- Clean Architecture

### Integrante
- Darling Báez

### Asignatura
Seminario de Proyecto II (ISW-411)

### Objetivo
Permitir a los usuarios programar, modificar, cancelar y consultar citas médicas desde una aplicación móvil segura y eficiente.
